# AI Finance Assistant（智能个人财务管理系统）

## 1. 项目简介

AI Finance Assistant 是一个全栈智能财务管理系统，结合 AI能力 + 个人消费分析 + 语音记账 + 数据可视化，帮助用户实现自动化财务管理与消费优化建议。

系统支持：

- 消费记录管理
- AI 自动分类消费
- 月度支出预测
- 个性化省钱建议
- 语音记账（中英文）
- 财务数据分析与可视化
- 用户偏好设置

## 3. 核心功能

### 3.1 Dashboard 总览
- 本月消费总览
- 分类统计
- 支出趋势图
- 财务健康概览

---

### 3.2 账单管理
- 添加 / 编辑 / 删除消费记录
- 分类管理
- 时间筛选
- 金额统计

---

### 3.3 AI 财务顾问
- 自动分析消费结构
- 提供节省建议
- 检测异常消费
- 生成月度总结报告

---

### 3.4 AI 自动分类
- 输入消费记录
- 自动识别类别（餐饮 / 交通 / 购物等）
- 支持自然语言输入

---

### 3.5 语音记账
- 语音转文字
- 自动结构化消费记录
- 支持中文 / 英文

---

### 3.6 财务分析
- 月度趋势分析
- 分类占比
- 消费变化对比

---

### 3.7 用户偏好系统
- 月预算设置
- 消费习惯记录
- 个性化建议优化
